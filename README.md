📖 Sobre o Projeto

O Bot Sergoias é uma ferramenta de automação desenvolvida em Node.js que interage com páginas web para resolver desafios automaticamente.

O bot:

- Detecta perguntas no DOM
- Coleta alternativas
- Envia para uma IA
- Seleciona a resposta automaticamente
- Avança para a próxima questão

Esse projeto foi criado como experimento de automação + integração com IA.

---

⚙️ Tecnologias

- Node.js
- Puppeteer
- Groq API
- Llama 3

---

🚀 Instalação

Clone o repositório:

git clone https://github.com/lucxslv/Bot-Sergoias.git

Entre na pasta:

cd Bot-Sergoias

Instale as dependências:

npm install

---

🔑 Configuração

Crie um arquivo ".env" na raiz do projeto:

GROQ_API_KEY=sua_api_key_aqui

---

▶️ Executando o Bot

node bot.js

O bot irá:

1. Abrir o navegador automaticamente
2. Acessar a plataforma
3. Identificar perguntas
4. Consultar a IA
5. Selecionar respostas
6. Avançar entre as questões
