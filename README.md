🤖 Bot Sergoias

Bot automatizado desenvolvido para interagir com páginas web e auxiliar na resolução de desafios de forma automática.

🚀 Funcionalidades

- Acessa automaticamente a plataforma
- Detecta perguntas e alternativas
- Utiliza IA para escolher respostas
- Navega entre questões automaticamente
- Automatiza tarefas repetitivas

🛠️ Tecnologias utilizadas

- Node.js
- Puppeteer – automação de navegador
- API de IA (Groq / Llama) – para análise das perguntas
- JavaScript

📦 Instalação

Clone o repositório:

git clone https://github.com/lucxslv/Bot-Sergoias.git
cd Bot-Sergoias

Instale as dependências:

npm install

⚙️ Configuração

Crie um arquivo ".env" na raiz do projeto:

GROQ_API_KEY=Sua_API_Key_aqui

▶️ Executando o bot

node bot.js

O bot irá:

1. Abrir o navegador
2. Acessar a plataforma
3. Detectar as perguntas
4. Consultar a IA
5. Marcar a resposta automaticamente
